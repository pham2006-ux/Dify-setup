# Dify ナレッジベース 自動同期 セットアップガイド（Googleスプレッドシート）

スプレッドシートの内容を毎日自動でDifyナレッジに同期する仕組みです。

---

## 設定値（コピペ用）

```
DIFY_API_KEY    : dataset-xAZAe3XcDG0RdDHXyFtFgRb7
DIFY_BASE_URL   : https://api.dify.ai/v1
DIFY_DATASET_ID : 4e5d0579-ee21-443d-b6d4-3e604b9fdd83
```

---

## セットアップ手順

### Step 1: GASスクリプトを追加する

1. ナレッジのスプレッドシートを開く
2. メニュー →「拡張機能」→「Apps Script」
3. 下にある **コード全体** をコピーして貼り付け
4. 保存（💾）

### Step 2: スクリプトプロパティを設定する

1. GASエディタ左メニュー → ⚙️「プロジェクトの設定」
   2.「スクリプト プロパティ」→「スクリプト プロパティを追加」
2. 以下の3つをそれぞれ入力：

**1つ目**

```
プロパティ: DIFY_API_KEY
値:         dataset-xAZAe3XcDG0RdDHXyFtFgRb7
```

**2つ目**

```
プロパティ: DIFY_BASE_URL
値:         https://api.dify.ai/v1
```

**3つ目**

```
プロパティ: DIFY_DATASET_ID
値:         4e5d0579-ee21-443d-b6d4-3e604b9fdd83
```

4.「スクリプト プロパティを保存」をクリック

### Step 3: 接続テスト

1. GASエディタ上部のドロップダウンで `testConnection` を選択
2. ▶ 実行
3. 実行ログで確認：

```
HTTP Status: 200   → ✅ 成功
HTTP Status: 401   → ❌ APIキーが違う
HTTP Status: 404   → ❌ Dataset IDが違う
```

### Step 4: 手動同期テスト

1. ドロップダウンで `syncKnowledgeToDify` を選択
2. ▶ 実行
3. 実行ログで以下が出れば成功：

```
=== Difyナレッジベース同期開始 ===
--- 既存ドキュメントの削除を開始 ---
--- 削除完了: ○件 ---
--- ドキュメント登録を開始 ---
  ✅ 登録成功: シート名1
  ✅ 登録成功: シート名2
--- 登録完了: ○件 ---
=== 同期完了 ===
```

4. Difyのナレッジ画面を開いてドキュメントが更新されているか確認

### Step 5: 毎日自動実行を設定する

1. GASエディタ左メニュー → ⏰「トリガー」
2. 右下の「+ トリガーを追加」
3. 設定：

```
実行する関数:                   syncKnowledgeToDify
デプロイ:                       Head
イベントのソース:               時間主導型
時間ベースのトリガーのタイプ:   日付ベースのタイマー
時刻:                           午前4時〜5時
```

4.「保存」→ 完了 🎉

## 貼り付けるコード（dify_knowledge_sync.gs の中身）

```javascript
// ============================================================
// Dify ナレッジベース 自動同期 GAS スクリプト
// ============================================================

function getConfig() {
  var props = PropertiesService.getScriptProperties();
  return {
    apiKey: props.getProperty("DIFY_API_KEY"),
    baseUrl: props.getProperty("DIFY_BASE_URL") || "https://api.dify.ai/v1",
    datasetId: props.getProperty("DIFY_DATASET_ID")
  };
}

function syncKnowledgeToDify() {
  var config = getConfig();
  if (!config.apiKey || !config.datasetId) {
    Logger.log("❌ エラー: スクリプトプロパティに DIFY_API_KEY と DIFY_DATASET_ID を設定してください");
    return;
  }
  Logger.log("=== Difyナレッジベース同期開始 ===");
  Logger.log("対象Dataset: " + config.datasetId);
  deleteAllDocuments(config);
  registerDocumentsFromSheet(config);
  Logger.log("=== 同期完了 ===");
}

function deleteAllDocuments(config) {
  Logger.log("--- 既存ドキュメントの削除を開始 ---");
  var page = 1;
  var hasMore = true;
  var deletedCount = 0;
  while (hasMore) {
    var listUrl = config.baseUrl + "/datasets/" + config.datasetId + "/documents?page=" + page + "&limit=100";
    var listResponse = UrlFetchApp.fetch(listUrl, {
      method: "get",
      headers: { "Authorization": "Bearer " + config.apiKey },
      muteHttpExceptions: true
    });
    var listData = JSON.parse(listResponse.getContentText());
    if (!listData.data || listData.data.length === 0) {
      hasMore = false;
      break;
    }
    for (var i = 0; i < listData.data.length; i++) {
      var doc = listData.data[i];
      var deleteUrl = config.baseUrl + "/datasets/" + config.datasetId + "/documents/" + doc.id;
      try {
        UrlFetchApp.fetch(deleteUrl, {
          method: "delete",
          headers: { "Authorization": "Bearer " + config.apiKey },
          muteHttpExceptions: true
        });
        deletedCount++;
        Logger.log("  削除: " + doc.name);
      } catch (e) {
        Logger.log("  ⚠️ 削除失敗: " + doc.name + " - " + e.message);
      }
      Utilities.sleep(500);
    }
    page = 1;
    if (listData.data.length < 100) { hasMore = false; }
  }
  Logger.log("--- 削除完了: " + deletedCount + "件 ---");
}

function registerDocumentsFromSheet(config) {
  Logger.log("--- ドキュメント登録を開始 ---");
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheets = ss.getSheets();
  var registeredCount = 0;
  for (var s = 0; s < sheets.length; s++) {
    var sheet = sheets[s];
    var sheetName = sheet.getName();
    if (sheet.isSheetHidden()) {
      Logger.log("  スキップ（非表示）: " + sheetName);
      continue;
    }
    var data = sheet.getDataRange().getValues();
    if (data.length <= 1) {
      Logger.log("  スキップ（データなし）: " + sheetName);
      continue;
    }
    var csvText = data.map(function(row) {
      return row.map(function(cell) {
        var str = String(cell);
        if (str.indexOf(",") >= 0 || str.indexOf("\n") >= 0 || str.indexOf('"') >= 0) {
          str = '"' + str.replace(/"/g, '""') + '"';
        }
        return str;
      }).join(",");
    }).join("\n");
    var createUrl = config.baseUrl + "/datasets/" + config.datasetId + "/document/create-by-text";
    var payload = {
      "name": sheetName,
      "text": csvText,
      "indexing_technique": "high_quality",
      "process_rule": { "mode": "automatic" }
    };
    try {
      var response = UrlFetchApp.fetch(createUrl, {
        method: "post",
        headers: {
          "Authorization": "Bearer " + config.apiKey,
          "Content-Type": "application/json"
        },
        payload: JSON.stringify(payload),
        muteHttpExceptions: true
      });
      var statusCode = response.getResponseCode();
      if (statusCode === 200 || statusCode === 201) {
        registeredCount++;
        Logger.log("  ✅ 登録成功: " + sheetName);
      } else {
        Logger.log("  ❌ 登録失敗: " + sheetName + " (HTTP " + statusCode + ")");
        Logger.log("     レスポンス: " + response.getContentText().substring(0, 200));
      }
    } catch (e) {
      Logger.log("  ❌ 登録エラー: " + sheetName + " - " + e.message);
    }
    Utilities.sleep(1000);
  }
  Logger.log("--- 登録完了: " + registeredCount + "件 ---");
}

function testConnection() {
  var config = getConfig();
  if (!config.apiKey || !config.datasetId) {
    Logger.log("❌ スクリプトプロパティが未設定です");
    return;
  }
  var url = config.baseUrl + "/datasets/" + config.datasetId + "/documents?page=1&limit=5";
  var response = UrlFetchApp.fetch(url, {
    method: "get",
    headers: { "Authorization": "Bearer " + config.apiKey },
    muteHttpExceptions: true
  });
  Logger.log("HTTP Status: " + response.getResponseCode());
  Logger.log("Response: " + response.getContentText().substring(0, 500));
}
```

---

## 注意事項

- 全削除→全登録方式のため、同期中の数秒間はナレッジが空になります。深夜帯での実行を推奨。
- 非表示シートとデータ1行以下のシートは自動スキップされます。
- GASの無料枠では1日90分の実行時間制限がありますが、通常の同期では問題になりません。
