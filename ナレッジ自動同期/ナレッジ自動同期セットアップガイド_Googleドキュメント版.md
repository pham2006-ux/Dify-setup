# Dify ナレッジ自動同期 セットアップガイド（Googleドキュメント版）

複数のGoogleドキュメントの内容を毎日自動でDifyナレッジに同期する仕組みです。

---

## セットアップ手順

### Step 1: GASプロジェクトを新規作成

1. Google Driveを開く
2.「新規」→「その他」→「Google Apps Script」
3. プロジェクト名を「Difyナレッジ同期」などに変更

※ スプレッドシート版と違い、独立したGASプロジェクトとして作成します

### Step 2: コードを貼り付け

1. `dify_knowledge_sync_googledocs.gs` の中身を全てコピー
2. GASエディタの `コード.gs` に貼り付けて保存

### Step 3: スクリプトプロパティを設定

1. GASエディタ左メニュー → ⚙️「プロジェクトの設定」
2.「スクリプト プロパティを追加」で以下を追加

**Dify接続情報（3つ）**

```
プロパティ: DIFY_API_KEY
値:         （ナレッジのAPIキー）
```

```
プロパティ: DIFY_BASE_URL
値:         https://api.dify.ai/v1
```

```
プロパティ: DIFY_DATASET_ID
値:         （ブラウザURLの datasets/ と /documents の間の部分）
```

**Googleドキュメント（5つ）**

値の書き方: `ドキュメントID:表示名` （コロンで区切る）

```
プロパティ: DOC_1
値:         ドキュメントID:IT・システム関連FAQ
```

```
プロパティ: DOC_2
値:         ドキュメントID:人事・労務FAQ
```

```
プロパティ: DOC_3
値:         ドキュメントID:経理・精算FAQ
```

```
プロパティ: DOC_4
値:         ドキュメントID:総務・施設FAQ
```

```
プロパティ: DOC_5
値:         ドキュメントID:社内規定FAQ
```

**ドキュメントIDの確認方法:**
```
GoogleドキュメントのURL:
https://docs.google.com/document/d/1AbCdEfGhIjKlMnOpQrStUvWxYz/edit
                                   ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                   ↑ この部分をコピー
```

3.「スクリプト プロパティを保存」

### Step 4: 接続テスト

1. ドロップダウンで `testConnection` を選択
2. ▶ 実行（初回は権限の許可が必要）
3. 実行ログで確認:

```
HTTP Status: 200   → ✅ 成功（次のStepへ）
HTTP Status: 401   → ❌ APIキーが違う
HTTP Status: 404   → ❌ Dataset IDが違う
```

### Step 5: 手動同期テスト

1. ドロップダウンで `syncDocsToDify` を選択
2. ▶ 実行
3. 実行ログで確認:

```
=== Difyナレッジ同期開始（Googleドキュメント版）===
--- 既存ドキュメントの削除を開始 ---
--- 削除完了: 0件 ---
--- ドキュメント登録を開始 ---
  ✅ 登録成功: IT・システム関連FAQ（1234文字）
  ✅ 登録成功: 人事・労務FAQ（2345文字）
  ✅ 登録成功: 経理・精算FAQ（1567文字）
  ✅ 登録成功: 総務・施設FAQ（890文字）
  ✅ 登録成功: 社内規定FAQ（3456文字）
--- 登録完了: 5件 ---
=== 同期完了 ===
```

4. Difyのナレッジ画面を開いて5つのドキュメントが登録されているか確認

### Step 6: 毎日自動実行を設定

1. GASエディタ左メニュー → ⏰「トリガー」
2.「+ トリガーを追加」
3. 設定:

```
実行する関数:                   syncDocsToDify
デプロイ:                       Head
イベントのソース:               時間主導型
時間ベースのトリガーのタイプ:   日付ベースのタイマー
時刻:                           午前4時〜5時
```

4.「保存」→ 完了 🎉

---

## ドキュメントの追加・削除

コードを触る必要はありません。スクリプトプロパティだけで管理できます。

```
追加したい → DOC_6 を新規追加するだけ（最大20個まで）
削除したい → そのDOC_X を削除するだけ
変更したい → DOC_X の値を書き換えるだけ
```

---

## 注意事項

- 全削除→全登録方式のため、同期中の数秒間はナレッジが空になります
- 深夜帯での自動実行を推奨
- GASからGoogleドキュメントを読む際、初回に権限の許可が必要です
- テキスト部分のみ同期されます（画像・表の書式は含まれません）
