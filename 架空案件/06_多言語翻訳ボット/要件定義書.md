# 要件定義書：多言語ビジネス翻訳ボット

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| **プロジェクト名** | 多言語ビジネス翻訳ボット |
| **Difyアプリ種別** | チャットフロー（advanced-chat） |
| **対象ユーザー** | 海外拠点との連携がある全社員 |
| **解決する業務課題** | ビジネスメール・資料の翻訳時に専門用語やトーンの統一が困難 |

## 2. 機能要件

### 2.1 対応言語
日本語 ↔ 英語 / 中国語 / 韓国語 / ベトナム語 / タイ語

### 2.2 翻訳モード
| モード | 説明 |
|--------|------|
| ビジネスメール | 丁寧なビジネストーン維持 |
| 技術文書 | 専門用語の正確な訳出 |
| カジュアル | 社内チャット向け |

### 2.3 処理フロー
1. コードノードで入力テキストの言語を検出
2. LLMで翻訳実行（翻訳モード指定）
3. 原文と訳文を対照形式で出力

## 3. ワークフロー構成

```
開始 → コード（言語検出・モード解析）→ LLM（翻訳実行）→ 回答
```

## 4. モデル設定

| 項目 | 設定値 |
|------|--------|
| **モデル** | gemini-2.5-flash |
| **Temperature** | 0.3 |
| **メモリ** | 有効 / ウィンドウサイズ10 |
