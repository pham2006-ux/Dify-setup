# 要件定義書：新人オンボーディングガイドボット

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| **プロジェクト名** | 新人オンボーディングガイドボット |
| **Difyアプリ種別** | チャットフロー（advanced-chat） |
| **対象ユーザー** | 新入社員・中途入社社員 |
| **解決する業務課題** | 新入社員が入社手続き・システム設定で同じ質問を繰り返し、人事・IT部門の対応工数が増加 |

## 2. 機能要件

### 2.1 対応範囲
| カテゴリ | 対応内容 |
|----------|---------|
| 入社手続き | 必要書類、提出期限、社会保険手続き |
| IT環境 | PC設定、メール設定、VPN接続、アカウント作成 |
| 社内制度 | 勤怠システム、経費精算、福利厚生 |
| オフィス | 入館証、座席、会議室、社内施設案内 |
| 連絡先 | 各部署の窓口、緊急連絡先 |

### 2.2 知識検索（ナレッジベース）
- オンボーディングマニュアルをナレッジベースに格納
- ユーザーの質問に対して関連するマニュアル情報を検索し回答

## 3. ワークフロー構成

```
開始 → 知識検索（オンボーディングナレッジ）→ LLM（ガイド回答生成）→ 回答
```

## 4. モデル設定

| 項目 | 設定値 |
|------|--------|
| **モデル** | gemini-2.5-flash |
| **Temperature** | 0.3 |
| **メモリ** | 有効 / ウィンドウサイズ10 |
